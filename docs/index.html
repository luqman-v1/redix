Redix

The next-generation Redis GUI for power users.

Fast, elegant, and lightweight. Built with Tauri v2, Rust, and Svelte 5.

⬇ Download View on GitHub

Screenshots

Browsing keys and inspecting values in the tree sidebar with tabbed detail views.

Rich value viewer with JSON auto-formatting and the live command console at the bottom.

Features

🎨

Modern & Premium UI

Deep dark glassmorphism aesthetic with slick animations and micro-interactions. Light mode included.

📂

Tabbed Interface

Work on multiple keys at once. Double-click any key from the sidebar to open it in a new tab.

🌳

Advanced Key Tree

Visualize deeply nested Redis keys (user:profile:123) in an intuitive tree structure.

📊

All Data Types

Rich, editable viewers for Strings, Hashes, Lists, Sets, and Sorted Sets. JSON auto-formatting included.

🖥️

Command Console

Monitor real-time commands with duration metrics, or type raw Redis commands directly.

📤

CSV Export

Instantly export Hash, List, Set, or Sorted Set data to .csv for external analysis.

⚡

Blazing Fast

Native performance powered by Tauri v2 and Rust — no Electron bloat, minimal memory footprint.

💻

Cross-Platform

Available for macOS, Windows, and Linux. One codebase, native feel on every OS.

Tech Stack

Svelte 5 Vite Tauri v2 Rust pnpm
